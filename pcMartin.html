La programmation de jeux sur PC

À la différence du développement sur consoles, les configurations pour deux joueurs donnés ne sont pas les mêmes. Il faut donc prévoir différents réglages graphiques permettant d’exploiter les possibilités offertes entre différents PC, qu’il soit plus puissant ou plus modeste en terme de configuration. Le prix de développement sur PC est plus faible car il n’est pas nécessaire de posséder une license pour vendre son jeu contrairement aux consoles et aux mobiles. Il existe une accessibilité différente en fonction des OS utilisés, on aura donc beaucoup plus de jeux présents sur l’OS windows que sur son concurrent Mac, ou encore linux. La programmation de jeu se fait beaucoup par l’intermédiaire de moteurs de jeux tels que Unity ou Unreal Engine, qui permettent la programmation d’un jeu via cette plateforme qui va structurer sa programmation en l’accompagnant grâce à une interface.

Les jeux en flash

On retrouve également pour la programmation web le module flash d'extension .swf. Il représente des données intéractive entre les pages web dans un format fermé de données d'animations. Ce plugin adobe flash n'est pas seulement nécessaire pour les jeux vidéo, mais on le retrouve aussi beaucoup dans l'animation vectorielle de sites. Les jeux flashs sont souvent liés aux jeux en HTML5 et en JS, deux langages web avec utilisation de la programmation orientée objet auniveau du JS qui est l'un des langages les plus utilisés.

Les jeux PC

Sur cette dernière console sortie par Nintendo il nous est également possible de créer des applications et donc des jeux vidéos. Le langage de programmation généralement utilisé sur consoles est le C++, il est également possible de coder en C# en utilisant des moteurs de jeux qui vont se charger de le transformer en code natif.
Le C++ est un langage qui permet d'utiliser la programmation orientée objet, procédurale et générique qui sont nécéssaires (notamment la prgrammation orientée objet) pour créer des jeux vidéos. Le C++ est en réalité la version suppérieure du Langage C (d'où son nom).
Le C# (sharp) est quant à lui un dérivé du C++, qui est très proche du java : langage utilisé pour créer des applications Androïd. Le C# reprend la syntaxe du langage Java et y ajoute certaines notions (surcharge des opérateurs, indexeurs, délégués), mais il est plus utilisé pour la programmation web (pour des applications) que por la créaion de jeux vidéos sur consoles.

La Xbox

C'est aujourd'hui grace au Xbox Live Creators Program que nous pouvons créer des jeux vidéo et les publier sur le Xbox Store le but était de favoriser les petits dévellopeurs et les studios indépendants. Sur la Xbox 360 par exemple les dévelloppeurs amateurs peuvent utiliser le langage C#, mais plus les plus grands dévellopeurs c'est un langage différent que leur a fournit Microsoft.
Il existe trois types de jeux sur Consoles: les jeux indépendants, les jeux d'arcarde. et les jeux officiels qui son les jeux fournis par Microsoft.
Pour les jeux indies, il faut utiliser XNA qui est une bibliothèque C# fournie par Microsoft. Pour les jeux d'arcade, il faut une licence qui donne accès à de meilleurs outils que XNA.
Dans les deux cas présentés ci-dessus l'utilisaion des librairies Xbox est payante et donne l'accès de la publication des jeux sur le store.
